Titel
Java Entwickler (m/w/d)
Beginn
07.10.2019
Einsatzort
Bad Homburg vor der Höhe
Ihre Aufgaben
Als Java Softwareentwickler gehen Sie eigenständig anfallenden Entwicklungs- und Weiterentwicklungstätigkeiten innerhalb des Java-Applikationsumfelds nach
Sie begleiten von der Anforderungsanalyse bis hin zur Umsetzung ganzheitlich den Entwicklungsprozess
Nach erfolgreicher Implementierung der umgesetzten Entwicklungen fungieren Sie als technischer Ansprechpartner für die jeweiligen Projekte
Eine saubere Codequalität liegt Ihnen selbst am Herzen, sodass für Sie eine „fehlerfreie“ Dokumentation selbstverständlich ist
Ihr Profil
Sie verfügen über eine abgeschlossene IT-Ausbildung zum Fachinformatiker Anwendungsentwicklung, ein abgeschlossenes Informatikstudium oder vergleichbare Qualifikationen
Ferner können Sie erste Berufserfahrung in der Entwicklung von Java Applikationen vorweisen, unter Nutzung von Java oder gar Java EE
Die Frameworks Spring und/ oder Hibernate sind Ihnen vertraut
Sie bringen ein ausgeprägtes Interesse an neuer Software mit und haben Spaß an der Findung spannender Lösungswege
Sehr gute Deutsch- und gute Englischkenntnisse runden Ihr Profil ab
